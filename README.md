# odin-recipes
Basic Dessert Recipe Website

Project Introduction:
Creating a basic dessert recipe website which will showcase a collection of recipes.
It will show users how to quickly make tasty dessert treats from start to finish,
using 5 ingredients or less, no more than 5.

Skills Demonstrated:
Through building this project I demonstrated the ability to structure
HTML documents and develop a project piece by piece, tackling one task
at a time rather than everything at once, which proved to be a more
effective and manageable approach for me.

I practiced using Git consistently, committing every time a piece of
HTML worked as expected.
This helped me understand that Git should only
be initialized once in the main project folder and not in any subfolder's,
as having two Git repositories in the same project caused conflicts and
the one in the subfolder had to be removed.

I also learned to search for what I needed when I was unsure, for example
finding the br tag for line breaks which was not covered in the coursework
but was needed to improve readability.

Overall this project demonstrated my ability to:
Write and structure HTML
Use Git for version control consistently
Create a GitHub repository and clone it using SSH
